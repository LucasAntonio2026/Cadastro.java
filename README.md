# 📋 Sistema de Cadastro em Java

Projeto simples desenvolvido em Java para cadastro de pessoas no terminal.

## 🚀 Funcionalidades

- Cadastrar pessoa
- Listar pessoas
- Remover pessoa
- Salvar em arquivo

## ▶️ Como executar

javac Cadastro.java  
java Cadastro
